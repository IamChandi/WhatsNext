# WhatsNext

WhatsNext is a premium, modern goal-tracking application for macOS built with SwiftUI and SwiftData. It features a streamlined interface, natural language parsing for quick entry, and powerful organization tools to help you focus on what truly matters.

## Key Features

- **Intuitive Dashboard**: A clean, focused view of your daily, weekly, and monthly goals.
- **Natural Language Parsing**: Create goals quickly by typing phrases like `!high Meeting tomorrow`.
- **Global Hotkey**: Access the Quick Entry field from anywhere with `Cmd + Shift + Space`.
- **Smart Reminders**: Automated notifications for morning planning and end-of-day reviews.
- **Multi-view Support**: Switch between Checklist and Kanban board views.
- **Archive System**: Keep your workspace clean by archiving completed or deferred goals.

## Architecture

The project follows a modular architecture designed for maintainability and scalability:

- **Models**: SwiftData-powered models with comprehensive documentation.
- **Views**: Highly modular SwiftUI views decomposed into reusable components and layout layers.
- **Services**: Dedicated services for notifications, natural language parsing, and global interactions.
- **Tests**: Unit tests covering core logic and parsing.

## Requirements

- **macOS 14.0+**
- **Xcode 15.0+**
- **Swift 5.9+**

## Getting Started

1. Clone the repository.
2. Open `WhatsNext.xcodeproj` in Xcode.
3. Build and run the `WhatsNext` target.
